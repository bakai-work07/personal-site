2 July 2025 / Game Design Making a 2D Game in Java #1

When I first learned Java in school, I didn’t exactly fall in love with it. The experience wasn’t helped by the fact that my professor was a bit of a CS purist. He was obsessed with textbook structure and strict design patterns. We were forced to write rigid, overly formal code that felt more like academic exercises than anything I actually wanted to build. I struggled with the syntax, and sometimes I felt like I was just writing code to satisfy a compiler, not to create something meaningful. But recently, I wanted to make a project I could see come to life: something visual, something I could play. So, I decided to take another shot at Java by building a 2D game.
